# osmosis-lp-tool

## Simple Python Tools for Osmosis AMM

### Installation

`pip3 install -r requirements.txt`

### Running

`./osmosis-lp-tool.py <bech32_address> [--csv]`

- `bech32_address` must be an osmo-prefixed Osmosis account.
- `--csv` is an optional flag for CSV output.
